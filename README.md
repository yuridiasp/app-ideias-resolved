## Requisitos
- NodeJS
- Editor de código

## Como usar

#### Faça o clone do repositório
```
git clone https://github.com/yuridiasp/app-ideias-resolved.git
```

#### Navegue até a pasta
```
cd app-ideias-resolved
```

#### Instale as dependências
```
npm install
```

#### Inicie a aplicação
```
npm run start:dev
```
